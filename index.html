Installation

Up and running in 4 steps.

The app uses a virtual COM port pair to receive data from Aronium. Here's exactly how to set it up.

1
Download & install the app
Download the latest Windows installer and run it. No admin required — the app installs in your user folder.
📦
AroniumPOS-Display-Client-Setup.exe
Windows 10 / 11 · x64
FREE
2
Create a virtual COM port pair
The app needs two virtual COM ports linked together. Aronium sends data on one, the app reads from the other. Install one of these free tools:
🔌
VSPE — Virtual Serial Port Emulator
by Eterlogic · Easiest to use · Recommended
⭐ EASY
🔧
com0com — Null-modem emulator
Open source · Advanced users
FREE
3
Configure Aronium POS
Tell Aronium to send customer display data to the first virtual port:
Settings → Devices
Customer Display → Enable ✓
COM Port → COM20
Use the left port of the pair you created. The app reads from the right port (e.g. COM21).
4
Launch & configure the app
Launch AroniumPOS Display Client. The setup wizard will ask you to select your two COM ports. Choose the pair you created.
# On phone/tablet — open browser and type:
http://YOUR-PC-IP:3000

# Example:
http://192.168.1.5:3000
The exact address is shown inside the app's Config panel (⚙️ top right).
Step-by-step guide

How to set up VSPE

VSPE is the easiest way to create a virtual COM pair. Follow these steps.

1
Download & install VSPE
Go to eterlogic.com, download VSPE and install it. Launch it as Administrator.
2
Create a new device
Click "Create new device" → choose "NullModem" → set Port A to COM20 and Port B to COM21.
3
Start the pair
Click Start. Both COM20 and COM21 will now appear in Windows Device Manager under Ports (COM & LPT).
4
Auto-start on boot
In VSPE settings, enable "Start with Windows" so your ports are always ready when the PC boots.
